# Smart Resume Analyzer

I built this project to make it easier to compare a resume with a job description. The application allows users to upload their resume in PDF format and paste a job description. It checks which skills are already present in the resume, identifies the missing skills, and shows a resume match score. This project helped me improve my knowledge of Python, Streamlit, and working with PDF files.

## Features
- Upload a resume in PDF format
- Paste a job description
- Find matching skills
- Show missing skills
- Calculate resume match score
- Suggest skills to improve

## Technologies Used
- Python
- Streamlit
- pypdf
- Git
- GitHub

## How to Run
1. Clone the repository.
2. Install the required packages using:
   pip install -r requirements.txt
3. Run the project using:
   streamlit run app.py
